# english-for-kids

Simple application for learning English words for kids. 

[**LIVE DEMO**](https://rolling-scopes-school.github.io/enthusiast17-JS2020Q3/english-for-kids/)

**Technologies:**
- TypeScript
- SCSS (BEM)
- Webpack 5

**Configs:**
- TS
- ESLint (AirBnB)

**Setup:**

Clone this repo and run the following commands:
```bash
  cd english-for-kids
  # to install packages
  npm install
  # to run website
  npm run start
```
